Python Aula 1


O que é Python?

  • Python é uma linguagem de programação de alto nível, interpretada, de propósito geral.
  • É uma linguagem popular para uma variedade de aplicações, incluindo ciência de dados, aprendizado de máquina, web desenvolvimento, e automação.

Por que aprender Python?

  • Python é uma linguagem relativamente fácil de aprender, mesmo para iniciantes. Possui uma grande comunidade online que oferece recursos e suporte.
  • É uma linguagem versátil que pode ser usada para uma variedade de aplicações.



Instalação do Python

  • O Python está disponível para Windows, macOS e Linux.
  • O processo de instalação é simples e direto.



Hello, World!

  • O programa "Hello, World!" é um programa simples que imprime a mensagem "Hello, World!" na tela.
  • É um bom primeiro programa para aprender a escrever código Python.

Além desses tópicos, os alunos também podem aprender sobre os seguintes conceitos básicos de Python:


Variáveis

  • As variáveis são usadas para armazenar dados.
  • Elas são nomeadas com uma letra ou um símbolo, seguido de um ou mais números ou letras.

Tipos de dados

  • Existem diferentes tipos de dados em Python, como números, strings e booleanos.
  • Cada tipo de dados tem suas próprias propriedades e comportamentos.

Operadores

  • Os operadores são usados para realizar cálculos e comparações.
  • Existem diferentes tipos de operadores em Python, como operadores aritméticos, operadores relacionais e operadores lógicos.

Comentários

  • Os comentários são usados para explicar o código.
  • Eles são ignorados pelo interpretador Python.

Após uma primeira aula de Python, os alunos devem ter uma compreensão básica da linguagem e ser capazes de escrever programas simples. Eles também devem estar familiarizados com os conceitos básicos de programação, como variáveis, tipos de dados, operadores e comentários.

Sobre o Projeto:

Este projeto foi feito para fins de estudo, desenvolvido pra cadastrar clientes de empresas substituindo assim o uso de planilhas Excel. Cadastro, visualização, pesquisa, alteração e remoção de registro. Conta com registro de pessoa física e da empresa na qual reside, podemos também adicionar imagen ao registro. usando banco de dados SQLite3. e com a possibilidade de exportar esses registros em formato excel.

Tecnologias e bibliotecas utilizadas.

Ao iniciar o projeto instale todas as dependencias contidas em requirements.txt

pip install -r requirements.txt
  • Pillow
  • API ViaCep
  • SQLite3
  • Cep gerado para fins de teste pelo site: https://www.4devs.com.br/gerador_de_cep
  • Cpf gerado para fins de teste pelo site: https://www.4devs.com.br/gerador_de_cpf
